# 🚀 this is for you (AI music playlist service)

> **"그날의 기분을 채팅으로 입력하면, 분위기에 꼭 맞는 맞춤형 노래 플레이리스트를 만들
어주는 AI 음악 추천 서비스."**

---

## 🎯 1. 기획 배경 및 해결 과제
* **현재 상황 및 문제점:**우리는매일다른감정과분위기를경험하지만, 그때그때어울리는음악을직접찾는일은 번거롭다. 기존 스트리밍 서비스의 추천은 청취 이력 기반이라, "지금 비 오는 날의 차분한 기분"처럼 순간의 감 정·맥락을 반영하지 못한다. 결국 사용자는 누군가에게 "이런 분위기 노래 좀 골라줘"라고 부탁하고 싶어진다.
* **우리가 제공할 해결책:**사용자가 채팅창에 오늘의 기분·상황·날씨를 자연어로 입력하면, AI가 감정을 분석해 **맞춤형플레이리스트**를즉시큐레이션한다. 추가로상황태그, AI 위로멘트, 날씨연동, 공유기능을통해단순추 천을 넘어 **"오늘의 감정 동반자"** 경험을 제공한다.


## 👤 2. 타겟 유저 페르소나
* **인구통계학적 특성:** 음악 스트리밍을 매일 1시간 이상 사용하는 10대 후반 ~ 20대 대학생/직장인. 모바일·웹
사용에 익숙하며 SNS 공유 친화적.
* **불편함:** 자신의 기분에 맞는 곳을 찾아 헤매는 시간이 길고, 알고리즘 추천이 비슷한 곡만 반복돼 새로움이 부족하다.
* **기대하는 것:** "내 기분을 이해받는" 따뜻한 큐레이션, 매일 다른 감정을 표현할 수 있는 공간, 친구와 공유할 수 있는 감성 콘텐츠.

## ⭐ 3. 핵심 기능 **(MVP + 추천 추가 기능)**
> 프론트엔드 환경에서 3주 내에 구현할 수 있는 핵심적인 기능 딱 3가지를 정의합니다.
  1. **[채팅 기반 감정 임력]**: 사용자는 메신저처럼 생긴 입력창에 오늘의 기분을 자유롭게 적는다. 말풍선 UI로 친근하
  게 대화하는 경험을 제공한다.
  2. **[AI 감정 분석]**: 입력된 텍스트에서 감정 키워드(기쁨/우울/설렘/차분 등)를 추출해 분위기 카테고리로 매핑한다.
  3. **[플레이리스트 추천]**: 분석 결과를 바탕으로 5~10곡의 맞춤 플레이리스트를 카드 형태로 보여주고, 마음에 드는 곡은내리스트에담을수있다.
  # 추천 추가 기능 #
  1. **상황태그선택**: "공부중 / 운동 / 비오는밤 / 출근길" 등클릭한번으로맥락보강.
  2. **AI 위로멘트**:추천과함께 "오늘하루도수고했어요☕" 같은짧은공감메시지노출.
  3. **날씨 연동**: OpenWeather API로 현재 위치 날씨를 자동 반영해 추천 정확도 향상.
  4. **플레이리스트 공유**: 추천 결과를 이미지/링크 카드로 변환해 SNS 공유 **→** 바이럴 유입.


## 🗺️ 4. 화면 설계 및 유저 플로우
* 사용자가 우리 웹사이트에 접속하여 목표를 달성하기까지의 클릭 흐름을 단계별로 작성합니다.
  1. **메인 페이지 접속**: 그라디언트 배경 + "오늘의 기분을 음악으로" 카피와 시작 버튼.
  2. **채팅페이지진입**:입력창에기분텍스트작성 (예: "비오는새벽에혼자카페에있는기분").
  3. **상황태그선택 (선택)**:칩형태태그 1~2개클릭으로맥락추가.
  4. **AI 분석 로딩**: "기분을 듣고 있어요..." 애니메이션 (1~2초).
  5. **결과페이지**:위로멘트 + 추천플레이리스트카드리스트표시.
  6. **곡담기 / 공유**: ♥ 버튼으로마이리스트추가, 공유버튼으로카드이미지생성.
